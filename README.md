# 설문조사 표본 공유 사이트 구현!
- 기존 번거로운 설문 조사 진행 방식을 개선하기 위해 구현 한 서비스  
- 기존 방식 : 주변 지인을 통해 설문 참여를 부탁함.  
- 개선 방안 : 하나의 사이트를 통해 설문 표본이 필요한 사람들을 모아 서로 설문 참여를 진행해 준다.  

## 제공 기능
- 회원 관리 서비스  
- 설문 관리 서비스  
- 관리자 서비스  

## 회원 관리
